An introduction to character physics

The PhysicsCharacter class (a subclass of PhysicsCollisionObject) provides a simplified physics model for characters.

The model incorporates the following assumptions:

  • The character’s shape is convex, and its orientation never changes.

  • The character remains upright even when jumping or falling: no sitting or lying down.

  • There are limits on steep a slope and how high a step the character can climb.

CharacterControl

To associate a character with a Spatial, there’s CharacterControl, a physics control. The PhysicsCharacter is created after the control is added to a Spatial.

In a CharacterControl, the physics control and its collision object are not the same object!

To access the collision object of a control, use characterControl.getCharacter(). To access the control of a collision object, use (CharacterControl) character.getUserObject().
